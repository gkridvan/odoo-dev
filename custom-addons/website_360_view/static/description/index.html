Website 360° View

Interactive panoramic product images for your e-commerce

Features

  • Upload equirectangular panorama images per product
  • Or link to externally hosted panoramas via URL
  • Interactive pan, zoom, and auto-rotate
  • Toggle between 360° and standard product images
  • Admin enable/disable per product
  • Multi-scene support with hotspots (advanced)
  • Fully responsive design
  • Powered by Pannellum (MIT licensed)

Usage

  1. Install the module
  2. Go to a Product → 360° View tab
  3. Enable 360° View
  4. Upload an equirectangular image or enter a URL
  5. Visit the product page on your website
